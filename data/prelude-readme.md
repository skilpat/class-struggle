# Instances visible in a couple representative Haskell modules

The following data shows how many type class instances are exposed by a couple representative Haskell modules:
the `Prelude` module, defined in the `base` package, and the `Data.Map` module, defined in the `containers` package.

* Haskell implementation: ghc-7.8.4
* All other packages: [LTS Haskell 2.3](http://www.stackage.org/snapshot/lts-2.3) from [Stackage](http://www.stackage.org/)
* Calculated using the code at https://github.com/skilpat/class-struggle
```haskell
import ReadWorlds
ctx <- buildCtx "<cabal-sandbox-path>" ["base", "containers"]
printCtx ["Prelude", "Data.Map"] ctx
```
